# Lab 2 — Simple CI/CD Pipeline (GitHub Actions)

This lab contains a minimal Node.js + Express app, Jest tests, and a GitHub Actions workflow.

## Structure
- `src/app.js` — Express application
- `tests/app.test.js` — Jest + Supertest tests
- `.github/workflows/simple-ci.yml` — CI workflow (install, test, build)

## Quick start
```bash
npm install
npm test
npm run build
npm start
```
